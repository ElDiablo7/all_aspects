# GRACE-X Lead Engine System Pack

This pack is a production-ready planning and starter-build bundle for turning:

- allaspectspaving.co.uk
- allaspectsbuilding.co.uk

into a dual-site Google and lead-generation system powered by GRACE-X.

## What this pack contains

- `MASTER_DEV_PROMPT.md` — Anti-Gravity / developer master implementation brief
- `SYSTEM_ARCHITECTURE.md` — full architecture and logic map
- `SEO_ROADMAP.md` — phased Google domination plan
- `PAGE_GENERATOR_TEMPLATE.csv` — page inventory starter template
- `CONTENT_BRIEF_TEMPLATE.md` — page-by-page authoring brief
- `CHATBOT_SYSTEM_PROMPT.txt` — GRACE-X sales chatbot prompt
- `FORM_SPEC.md` — form fields, routing, validation, CRM logic
- `lead_engine_starter/` — starter HTML/CSS/JS components for form, chatbot shell, scoring, and routing
- `SITEMAP_BLUEPRINT.md` — site structure and internal linking map
- `NEXT_30_PAGES.md` — recommended first 30 pages to build
- `IMPLEMENTATION_CHECKLIST.md` — deployment checklist

## Important

This is a serious starter pack, not a fake placeholder zip.
It does **not** contain a full live production website for both brands, but it does contain the core logic, prompts, templates, and starter components needed to build it properly.

## Suggested order

1. Read `MASTER_DEV_PROMPT.md`
2. Review `NEXT_30_PAGES.md`
3. Build pages from `PAGE_GENERATOR_TEMPLATE.csv`
